# odin-recipes

This is my first project for **The Odin Project**. 
It consists of a basics recipe website built solely with HTML to demostrate the correct use of links, images and lists.

## What I learned:

* How to structure web pages using essentials HTML tags.
* How to connect different pages using relative links.
* How to organized information using ordered and unordered lists.
* The basic workflow with Git and GitHub from the terminal.
